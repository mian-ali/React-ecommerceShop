## Install
```
$ clone git@https://github.com/mian-ali/React-ecommerceShop.git
$ yarn
```

## Run Server
```
$ npm install -g json-server
Then
$ yarn server
or
$ json-server public/db.json --port 8000
```

## Run App
```
$ yarn start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
